# Portfolio - Ronny Gutierrez

Modern, minimal portfolio built with Astro + TailwindCSS.

## Features

- Dark theme with gradient accents
- i18n support (EN/ES)
- Fully responsive
- CSS-only animations
- Modular components

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## Customization

### Content

Edit translations in `src/i18n/en.ts` and `src/i18n/es.ts`.

### Assets

- Replace `public/cv.pdf` with your actual CV
- Replace `public/images/profile.svg` with your photo
- Update company logos in `public/logos/`

### Colors

Edit `tailwind.config.mjs` to customize the color palette.

## Tech Stack

- Astro 4
- TypeScript
- TailwindCSS 3
- Inter font (Google Fonts)
